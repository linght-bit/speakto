# 🚀 QUICK START — Минимально рабочая версия

## Запуск

Откройте `index.html` в браузере.

## Что вы увидите

1. **Зелёный экран** (фон)
2. **Текст "Game Loading..."** (заглушка)
3. **Консоль (F12)** показывает инициализацию

## Консоль должна показать

```
✅ ГОТОВО К ЗАПУСКУ! (10/10)
✓ Все 10 модулей загружены
```

## Основные файлы

```
/core/gameState.js      ← Одно состояние игры
/core/events.js         ← Система событий
/utils/getText.js       ← Тексты (i18n)
/config/gameConfig.js   ← Конфиг игры
/systems/               ← Системы (квесты, диалоги, инвентарь)
/ui/render.js           ← Рендеринг
/bootstrap.js           ← Запуск игры
```

## Правила архитектуры

✅ Все тексты в `/i18n/*.json`  
✅ Все данные в `/data/*.json`  
✅ Все состояние в `gameState.js`  
✅ Все события в `eventSystem`  
✅ Никаких глобальных переменных  

## Проблемы которые были исправлены

| Проблема | Решение |
|----------|---------|
| gameConfig не загружалась | Добавлен `window.gameConfig` |
| Модули не были глобальными | Все прикреплены к window |
| bootstrap падал без JSON | Graceful degradation |
| Системы требовали gameState | Добавлены проверки |
| Canvas ломался если нет элемента | Добавлена обработка ошибок |
